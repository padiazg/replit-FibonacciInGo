# Fibonacci
Non recursive approach


$$
f_{n}  =  
  \begin{cases} 
  0 & \text{if } n =  0  \\
  1 & \text{if } n = 1 \\
f_{n-1}+f_{n-2} & \text{if } n >=2
  \end{cases}
$$
